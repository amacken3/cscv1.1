# Nitro Packs – Card Shop Client v1.1

Card Shop Client is a front-end React application that simulates a small online trading card store.

It focuses on delivering a smooth user experience through clean UI design, responsive layouts, and functional shopping features.

The project demonstrates practical React skills including component structuring, state management with Context API, page routing, and interactive UI behavior.

---

##  Tech Stack

- **React (Create React App)**
- **React Router**
- **Context API** – global state management
- **JavaScript (ES6)**
- **Custom CSS** for layout, responsiveness, and animations

---

##  Features

- Interactive product grid with hover animations  
- Fully responsive layout for mobile & desktop  
- Add/remove items from the cart  
- Increase or decrease item quantities  
- Real-time auto-calculated totals  
- Checkout page with mock payment form  
- Success screen with simulated order completion  
- Smooth fade-in transitions between pages  
- Sticky navigation bar with live cart item counter
